## versions/baby-mode
This branch is dedicated to a version of our code called "Baby Mode." It"s more gentle and tuned-down for stuff such as drivetrain velocity and shooter angular velocity.

It's mainly used for the purpose of showcases and areas where we can't run our robot in full power for safety.
