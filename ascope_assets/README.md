1. Extract [`Robot_StuyPlus.zip`](https://github.com/StuyPulse/StuyPlus-2026/blob/main/ascope_assets/Robot_StuyPlus.zip)
2. Go to the MenuBar and go to `App > Use Custom Assets Folder`, and select this directory [`ascope_assets`](https://github.com/StuyPulse/StuyPlus-2026/tree/main/ascope_assets),
3. Go to the MenuBar and go to `File > Import Layout`, and select [`AdvantageScope-Layout.json`](https://github.com/StuyPulse/StuyPlus-2026/blob/main/ascope_assets/AdvantageScope-Layout.json) from this directory
4. Simulate Robot Code
5. Go to the MenuBar and go to `File > Connect to Simulator > NetworkTables 4`. Do <b><u>NOT</u></b> select `NetworkTables 4 (Advantage Kit)` as we currently don't use AdvantageKit.
